# 아이스 브레이킹 게임 서비스 | 얼음땡

![썸네일](https://github.com/bsideproject/15_12_FE/assets/92833235/00540b79-cea4-4db8-af91-20efce6876d6)

> 꽁꽁 얼어붙은 어색함을 부숴줄 아이스 브레이킹 도구 모음

## Stacks

- **FrameWork** : Next.js + TypeScript
- **Styling** : Tailwind CSS
- **Store** : Recoil
- **API** : React Query
- **WebSocket** : stomp, SockJS

## Requirements

- [Node.js 18.15.0](https://nodejs.org/ca/blog/release/v18.15.0)
- [yarn 1.22.10](https://github.com/yarnpkg/yarn/releases/tag/v1.22.10)

## Install

```
$ git clone https://github.com/bsideproject/15_12_FE.git
$ cd 15_12_FE
$ yarn install
$ yarn dev
```

## Branch Naming Rule

- **main** : 메인 브랜치
  <!-- **develop** -->
- **feature** : 기능 개발
- **hotfix** : 버그 수정

## Commit Convention

- [Feat] : 새로운 기능 추가
- [Fix] : 버그 수정
- [Update] : Fix와 달리 원래 정상적으로 동작했지만 보완의 개념
- [Design] : CSS 등 사용자 UI 디자인 변경
- [Style] : 코드 포맷 변경, 코드 의미에 영향을 주지 않는 변경 사항
- [Docs] : 문서 수정
- [Test] : 테스트 코드 추가 (프로덕션 코드 변경 X)
- [Refactor] : 프로덕션 코드 리팩토링
- [Comment] : 필요한 주석 추가 및 변경
- [Chore] : 빌드 부분 혹은 패키지 매니저 수정사항 (프로덕션 코드 변경 X)
- [Rename] : 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우
- [Remove] : 파일을 삭제하는 작업만 수행한 경우
